# Excel Sales Dashboard Project

## 📌 Project Overview
This project analyzes sales data using Microsoft Excel to create an interactive dashboard.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Formulas

## 📊 Dashboard Features
- Total revenue calculation
- Revenue by city
- Revenue by category
- Product-wise revenue analysis

## 🔍 Key Insights
- Ottawa generated the highest revenue
- Electronics was the best-performing category
- Laptop generated the highest product revenue

## 🚀 Skills Demonstrated
- Excel formulas
- Pivot tables
- Data visualization
- Dashboard creation
- Business insight generation

## 📈 Conclusion
This project demonstrates how Excel can be used to analyze sales data and build a simple business dashboard.
